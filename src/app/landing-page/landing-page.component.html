¿Qué es NexTask?

NexTask es una aplicación web que nos permite crear listas de tareas o de pendientes de forma en la cual podamos tener una mejor organización de nuestras actividades.

¿Qué Ofrecemos?

Proporcionamos una aplicación web a aquellas personas mediante la cual puedan organizar sus pendientes en diferentes listas, además de esto, tener la posibilidad de visualizar las tareas.
